# AWYS Specs & Notes

This folder tracks feature specs, decisions, and progress for AI While You Sleeping.

## Structure

- `architecture.md` — System architecture overview and tech decisions
- `roadmap.md` — Current priorities and what's next
- `decisions.md` — Key decisions log with dates and reasoning
- Feature specs go in individual files: `feature-name.md`

## How to use

- Claude will reference these files for context on ongoing work
- Update specs as features are built or requirements change
- Log decisions so future sessions know WHY something was done a certain way
